# ERP Mobile App

Mobile application for ERP Platform using React Native + Expo.

## Features

- Product browsing
- Order tracking
- Real-time notifications
- Offline mode support
- GPS/Map integration
- Barcode scanning

## Installation

```bash
npm install
```

## Development

```bash
npm start
```

Then press `a` for Android, `i` for iOS, or `w` for web.

## Building

```bash
eas build --platform android
eas build --platform ios
```

## Testing

```bash
npm run test
```
