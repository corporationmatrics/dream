# ERP Mobile Admin App

Admin mobile application for ERP Platform using React Native + Expo.

## Features

- Dashboard with key metrics
- User management
- Analytics and reports
- Real-time data visualization
- Offline support
- Push notifications

## Installation

```bash
npm install
```

## Development

```bash
npm start
```

Then press `a` for Android, `i` for iOS, or `w` for web.

## Building

```bash
eas build --platform android
eas build --platform ios
```

## Testing

```bash
npm run test
```
