# ERP API - NestJS Backend

Main API gateway for the ERP Platform.

## Features

- User Management
- Product Catalog
- Orders & Inventory
- Authentication & Authorization
- Real-time WebSocket support
- RESTful API endpoints

## Installation

```bash
npm install
```

## Configuration

Copy `.env.example` to `.env` and update the configuration values.

## Running the app

```bash
# development
npm run start:dev

# production mode
npm run start:prod
```

## Testing

```bash
npm run test
npm run test:cov
npm run test:e2e
```

## API Documentation

Endpoints will be documented in the `/api/docs` route using Swagger.
