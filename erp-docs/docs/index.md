# Overview

Welcome to the ERP Platform Documentation!

## What is ERP Platform?

ERP Platform is a comprehensive, open-source enterprise resource planning system built with modern technologies including NestJS, Next.js, React Native, PostgreSQL, and more.

## Key Features

- **Multi-module architecture**: Accounting, Inventory, Orders, Users
- **Web & Mobile support**: Next.js web app + React Native mobile app
- **Real-time capabilities**: WebSocket support for live updates
- **Scalable infrastructure**: Kubernetes-ready with Docker support
- **Advanced analytics**: ML-powered predictions and insights
- **Enterprise-grade**: ACID compliance, backup & restore, monitoring

## Technology Stack

### Backend
- NestJS (Node.js API Framework)
- Spring Boot (Accounting Service)
- FastAPI (ML Services)

### Frontend
- Next.js 14 (Web Application)
- React Native + Expo (Mobile Apps)

### Infrastructure
- PostgreSQL (Primary Database)
- KeyDB (Cache Layer)
- Meilisearch (Full-text Search)
- MinIO (Object Storage)
- Kubernetes (Orchestration)
